# 🦷 Nirmal Chaow Dental Care

A modern, responsive, and user-friendly Dental Clinic website built using **React.js** and **Tailwind CSS**. The website provides patients with information about dental services, clinic details, appointment booking, testimonials, and contact information.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Piyushpandey09/dental-clinic.git
```

### Go to Project Folder

```bash
cd dental-clinic
```

### Install Dependencies

```bash
npm install
```

### Run Project

```bash
npm start
```

Application runs on:

```text
http://localhost:3000
```
